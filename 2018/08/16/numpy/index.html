Numpy入门

Numpy 的特点

  1. 擅长数值计算
  2. 足够高的运算性能，使用c语言编写
  3. 支持矢量计算运算

  4. 免费，开源

    # -*- coding: utf-8 -*-
from __future__ import unicode_literals

import datetime as dt
import numpy as np

n = 100000
A, B = [], []
C = []

def origin():
    for i in range(n):
        A.append(i**2)
        B.append(i**3)
    for a, b in zip(A, B):
        C.append(a + b)

def nums():
    A, B = np.arange(n)**2, np.arange(n) ** 3
    C = A + B

if __name__ == '__main__':

    start = dt.datetime.now()
    nums() # 1000ms
    print((dt.datetime.now() - start).microseconds)

    start = dt.datetime.now()
    origin() #10000ms
    print((dt.datetime.now() - start).microseconds)

Numpy的数组- ndarray

ndarray的特点

  1. Numpy中的数组是 ndarray 类类型的对象

    • 将实际数据和元数据分开存放，独立操作，以此提升性能
    • 元数据 是对数据的一种描述， 数据的类型，数据的维度，数据的大小

  2. Numpy 数组的元素类型必须相同 – 同质性

    • ndarray的元素类型不同时
    • ndarray的每个维度的数据数量不相同时
    • ndarray的元素会变为非同质对象
    • 非同质对象ndarray无法有效发挥Numpy优势，尽量避免使用

  3. Numpy的数组的元素可以通过 基于0 的下标单独访问

    • 按照维度的从高到低进行提取 i > j > k
    • c[i][j][k] == c[i,j,k]

  4. Numpy数组通过dtype和shape属性表示元素的类型和维度

    • 维度的是一个元组，按照从高到低的顺序排列每一维的大小

      1 2 3
      4 5 6
      shape = (2, 3) 2 行 3 列

      1 2 3

      4 5 6

      7 8 9
      shape = (3, 1, 3) 3页 1行 3 列

  5. numpy数组中 掩码

    1. 一个数组中全是 bool值

      a = [1,0,1,1,0]
      b = [1,2,3,4,5]
      b[a] = [1,3,4]

    2. 通过使用掩码筛选数组

ndarray的维度变化

  • 注意转换后，原数组是否改变
  • 维度转换，数据个数不会变化
  • 视图变维 （复制元数据进行改变）， 复制变维(复制实际数据进行改变), 就地变维（改变数据本身）

  • 视图变维后加上.copy()可变为复制变维

    ravel().copy() == flatten()

方法 说明 变维方式
.reshape(shape) 不改变数组元素，返回一个shape形状的数组，原数组不变 视图变维
.ravel() 无参数，返回数组的一维视图 视图变维
.resize(shape) 与.reshape()功能一致，但修改原数组 就地变维
.transpose 使用线性代数进行转置，转置视图，行列互换 视图转置
.T 与transponse一致，在二维数据及以上使用
.swapaxes(ax1,ax2) 将数组n个维度中两个维度进行调换
.flatten() 对数组进行降维，返回折叠后的一堆数组，原数组不变 复制变维

组合和拆分

不同数组之间的维度要相等

函数 说明 类型
.vstack() 堆叠数组，相当于将数组进行拼接，不改变维度 垂直组合
.vsplit(数组，份数) 对数组进行拆分，数组要被份数整除 垂直拆分
.hstack() 组合数组，相对于将数组进型拼接，不改变维度 水平组合
.hsplit() 对数据进行拆分 水平组合
.dstack() 可增加一个维度并旋转90度 深度组合
.dsplit() 与stack不同，不会发生维度的变化，转换后的结果不与dstack相反 深度拆分
.row_stack((上，下)) 一维数值的组合，与vstack（）相同 行组合
.column_stack((左，右)) 一维数值的组合，不与hstack（）相同 列组合 
# 深度组合和拆分
arr1 = np.arange(1,10).reshape(3,3)
arr2 = arr+10
arr3 = np.dstack((arr1,arr2))
# 将叠加后的数据拆分，拆分后唯独不变化
arr4,arr5 = np.dsplit(arr3,2)
# 使用transpose进行转换后，可得到原来的数据
arr4.T[0] == arr1 # True
