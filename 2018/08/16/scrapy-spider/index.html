Spider

  • spider类定义了如何爬取网站，包括怕去的动作，以及如何从网页中的内容提取结构化数据
  • spider类继承于Scrapy.spiders.Spider
  • parse 函数 是解析网页并返回信息的函数，parse接受response，返回包括 Item对象，dict，Request的可迭代容器

在命令行 执行 crawl spider后执行的流程

  1. 命令 crawl 执行 == > start_requests()执行，并且调用make_requests_from_url(url)
  2. make…(url) 获取start_requests传入的url，并返回request
  3. 默认情况下，parse函数接收make…(url)传入的request，进行进一步的操作
  4. 操作response解析数据，详见scrapy-selector

  5. 获取数据，传入item的类中

    from ..items import TencentspiderItem

def parse(self, response):
    ...
    item = TencentspiderItem()
    for description, url, location, release_time in \
    zip(descriptions, urls, locations, release_times):
        item['description'] = description
        item['url'] = url
        item['location'] = location
        item['release_time'] = release_time
        # print(item)
        yield item

  1. from_crawler 类方法 可获取settings.py中的配置

    @classmethod
def from_crawler(cls, crawler):
    #  获取settings 的配置

    mongo_uri = crawler.settings.get("MONGO_URI")
    mongo_db = crawler.settings.get('MONGO_DB')
    collection_name = crawler.settings.get('MONGO_COLLECTION')

    return cls(mongo_uri, mongo_db, collection_name)
