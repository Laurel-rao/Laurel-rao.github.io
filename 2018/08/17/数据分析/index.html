argmax/argmin 在一个数组中寻找最大值的下标，最小值的下标

数组的标准差 σ

np.std(S) ===> std
使用总体方差，除以n ，numpy使用默认 ddof = 0
使用样本方差，除以n-1 numpy.std(S, ddof=1)->std

  • 数学方程式
  • 样本: S = [a,b,c,d]
  • 均值: M = (a+b+c+d)/n
  • 离差: D = [a-M, b-M, c+M, d-M]
  • 离差方: Q = [(a-M)2 + (b-M)2…] ==>q = (a-M)**2
  • 方差: v = (q1 + q2 + q3…)/n
  • 标准差: std = sqrt(v)

  • 标准差也称为 方均根误差

    stanrd = sqrt((（样本值-均值）** 2 + 。。。)/n)
