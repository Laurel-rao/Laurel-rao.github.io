了解 Flask

flask搭建示例

from flask import Flask

app = Flask(__name__)

# 路由
@app.route('/')
def hello_world():
    return 'Hello world'

if __name__ == '__main__':
    app.run()

  1. app 为Flask 创建的对象，意思是导入本模块

    • 假如有多个模块，使用另一个文件写入 

      from app import app

if __name__ == '__main__':
    app.run()

  2. 路由使用route方法，装饰在 view 函数上，并且可以多个路由装饰在一个view上

    @app.route('/')
@app.route('/index')
def hello_world():
    return 'Hello world'

  3. return 返回给response

Flask 扩建

扩建路由 url

  • @app.route(‘/index/‘) ==== 普通URL

  • @app.route(‘/‘) ==== 传参

    @app.route('/<index>')
def hello_world(index):
    return 'Hello world %s'%index

  • @app.route(‘/index/‘, methods=[‘get’])

    1. methods 参数是一个列表

扩建views

  • ‘ ‘ 返回字符串
    • return ‘hello world’

  • redirect URL 反转用于重定向

    • url_for(‘函数名’)

      @app.route(‘/‘)
      def hello_world(index):

      return 'Hello world %s'%index

      @app.route(‘/login/‘)
      def login():

      return redirect(url_for('hello_world'))

  • render_template() 渲染模板

    • jinja2模板引擎

    • <a href=http://jinja.pocoo.org/docs/2.10/templates/>网站 

      {% ... %}    语句块开始
{% endxxx %} 语句块结束
{{ ... }}    变量值
    注释

    • render_template(‘xxx.html’, age=13)

    • 传入变量，函数， 语句
    • {% if age >10 %}
			  
      Hello {{ age }}!
      
			{% elif age = 10 %}
			  
      Hello {{ age + 10 }}!
      
			{% else %}
			  
      Hello, World!
      
			{% endif %}

    • 注意，在包内创建 templates 是唯一的方法，

    • 并且需要在运行模块文件的旁边或者在app包内

  • respon = make_response(render_template(…)) 响应对象

静态文件

- 创建文件夹static，与 templates 在同一路径下

- 直接填入路径

        <img src="/static/wangyiyun.PNG" alt="">

- 使用url_for 进行构造

request 对象

print(request.method)
print(request.path)
# post表单内容
print(request.form) # ImmutableMultiDict([])
print(request.get_data())   # json格式
print(request.get_json())   # 字典格式
# get请求内容
print(request.args) # ImmutableMultiDict([])
print(request.environ) # wsgi 的environ信息
print(request.cookie)

cookie

  • 设置cookie

    1. 构建响应对象 resp = make_response(render_template(…))

    2. 设置cookie resp.set_cookie(‘’, expires=xxx, max_age=3600 # 一个小时)

    3. return resp

  • 获得cookie

    1. request.cookies.get(‘xxx’)

  • 删除cookie

    1. resp = make_response(render_template(…))
    2. 第一种 resp.delete_cookie()
    3. 第二种 resp.set_cookie(xxx, expires=0)
    4. return resp

错误 和 日志

  • abort(401)

  • 设置自定义错误页面

    @app.errorhandler(404)
def page_not_found(error):
    return render_template('page_not_found.html'), 404

  • 日志

    app.logger.debug('A value for debugging')
app.logger.warning('A warning occurred (%d apples)', 42)
app.logger.error('An error occurred')

session

除了请求对象之外，还有第二个对象 session，

它允许您将特定于用户的信息从一个请求存储到下一个请求

这是在cookie的基础上实现的，并以加密方式对cookie进行签名

这意味着用户可以查看cookie的内容但不能修改它，除非他们知道用于签名的密钥

if __name__ == '__main__':
    # 设置 secret_key
    app.config['SECRET_KEY'] = os.urandom(16)
    app.run(debug=True, port=2766, host='0.0.0.0')

  • 设置session

    1. session[‘username’] = request.form[‘username’]

  • 获得session

    1. a = session[‘username’]

  • 删除session

    1. session.pop(‘username’, None)
