python 常用爬虫库

Posted on 2018-08-22 | In 爬虫

requests 库

  1. 发送请求

    • 基础方法 requests.request(method, url, **kwargs)

      • requests.request 是爬虫的基础方法，大部分请求都是对此方法进行封装
      • 使用示例
        response = requests.request('get', 'https://www.baidu..com')
print(response.text)

    • 常用方法

      • requests.get(url, params)
        • params 接受一个字典作为参数
        • 拼接到url后发送请求
      • requests.post(url, data, json)
        • data 和 json 都是传输数据，格式不一致

    • 13个可选参数

      • 较常用
        • headers
          • 定制一个请求头，会与原来的请求头进行一次合并，以用户定制的为准
          • 常用的请求头
            • User-Agent
            • referer
            • origin
        • cookies
          • cookies 可传入 字典或者 cookieJar对象
        • params
        • json
          • 使用 content-type: application/json 的请求头
          • 传入 字符串，列表，或字典
        • data
          • 使用 conten-type: application/x-www-form-urlencoded 的请求头
        • files
          • 使用content-type: multipart/form-data;boundary=… 的请求头，传送文件
          • 使用举例
            • rep = requests.get(‘http://www.baidu.com', files={‘pic’: open(“123.jpg”, ‘rb)})
        • timeout
          • 设置请求超时时间，超时引发异常
      • 其他高级用法, 具体是对证书和权限的控制
        • auth
        • verify
        • cert
        • stream
        • allow_redirects

  1. 获取本次请求的具体信息

    • rep = requests.get(“https://www.baidu.com")
    • rep.text
    • rep.content
    • rep.status_code
    • rep.json()
    • res = rep.request
      • res.headers
      • res.url
      • res.method
    • rep.cookies.get_dict()
    • rep.raise_for_status()
      • 当响应码不为 200 时，引发异常
