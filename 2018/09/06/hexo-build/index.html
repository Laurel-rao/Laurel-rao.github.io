hexo命令

命令 说明
hexo server 启动服务器
hexo d 部署网站
hexo g 生成静态文件
hexo clean
hexo d -g 生成静态页面之后，部署网站
hexo new ‘名称’

hexo 无法渲染时，会出现

  • TypeError: Cannot read property ‘utcOffset’ of null
  • 是因为时区设置不对， Asia/Shanghai，
  • 需要将 hexo 下 和 主题下的 _config.yml 的 timezone 设置正确
