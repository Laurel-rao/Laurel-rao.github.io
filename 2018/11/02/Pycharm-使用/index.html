Pycharm 使用

Posted on 2018-11-02 | In 常用软件操作

  1. 给常用的语句设置快捷口令

    1. Editor -> Live Templates ->

    2. 选择语言, 点击加号，选择Live Template

    3. 创建一个新的快捷口令(abbreviation: 缩写，缩略语， description: 描述，说明， template text: 这里就是代码了)

    4. 选择该快捷口令的使用区域，再点击 apply 就 ok了

    5. 旁边有一块区域可以定义变量，暂时没有研究

  2. pycharm 支持所有设置保存到一个文件上，这样不论是在哪里使用pycharm，只需要导入settings文件，就能保证使用一致了

    1. 导出settings文件

      1. 文件

      2. 远程仓库

    2. 导入文件

  1. 设置开机启动脚本

    1. 在Tools -> startup Tasks -> 点击 ➕
    2. 设置 脚本()

  2. 设置terminal 字体大小

    1. 找到 editor -> general -> console -> console commands history size
    2. 调整数值

  3. 查找相应设置使用搜索，快捷方便
