英语前缀后缀词根

Posted on 2018-11-19 | In 英语

后缀

  1. 名词后缀

    1. -er 加载动词后构成表示从事该动作的人或物，或者工具
    2. -or 与 er类似，大部分接在含有字母t的动词后
    3. -ion
      • 以t、s结尾的动词，直接加 -ion
      • 以 te 结尾的单词，去掉e，再加 -ion
      • 以 d 、de 结尾的动词，先改 d、de 为 s，再加 -ion
        attend (例外) - attention
      • 以 mit 结尾的动词，改 mit 为 miss， 再加上 -ion
      • 以辅音字母结尾(ts)除外的动词，常加 -ation 或 -ition
      • 以辅音字母加不发音字母 e(te 或 de 除外)结尾的动词，常 去掉 e再加-ation 或 -ition
      • 词干变化 再加 -tion
        • describe 描述 description 描述
    4. ment

  2. 形容词后缀

    1. -al

      • accident ==> accidental 意外的，偶然的
      • parent ==> parental 父母亲的

    2. -ous

      • danger ==> dangerous 危险的
        • humor ==> humorous 幽默的，可笑的

      1. -ful

        • care ==> careful 仔细的，小心的
        • fear ==> 鲜艳的，色彩艳丽的

      2. -less

        • care ==> careless
        • end ==> endless

      3. -y

        • cloud ==> cloudy 多云的，阴天的
        • fog ==> foggy 多雾的, 雾大的

      4. -able 名词加上able构成形容词,这种形容词意为“具有……特点的”

        • value ==> valuable
        • comfort ==> comfortable 舒适的
        • reason ==> reasonable 合理的

      5. -able 动词加上后缀 -able， 构成的形容词为 “能 。。。 的”， “可以(被) 。。。”， “适合于 。。。”， “值得 。。。 的”

        • enjoy ==> enjoyable 快乐的
        • count ==> countable 可数的

      6. -ive

        • 动词加后缀ive变成形容词
        • act ==> active 积极的，活跃的
        • create ==> creative 有创造力的，有创意的

前缀

  1. un-
  2. dis-
  3. ob-
  4. bi- 双的
  5. tri- 三个的
  6. bio- 生命
  7. trans- 穿过，越过
  8. pre- 预先，提前
  9. sub- 在 。。。 下
  10. multi- 多个
  11. con- 共同，一起
