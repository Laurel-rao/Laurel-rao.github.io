数据库基础概念

名词 英文 含义
数据 data 能够输入到计算机中并被识别处理的信息集合
数据结构 data structure 研究一个数据集合中数据之间关系的
数据库 database 按照数据结构，存储管理数据的仓库。数据库是在数据库
管理系统管理和控制下，在一定介质上的数据集合。
数据库管理系统 DBMS 管理数据库的软件，用于建立和维护数据库
数据库系统 DMS 由数据库和数据库管理系统，开发工具等组成的集合
数据库管理员 DBA(Administrator) 数据库管理人员
